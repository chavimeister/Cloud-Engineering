# Licorne Caca Fiesta

Static web app Angular standalone, volontairement tres kitch, avec un mini-jeu ou une licorne saute par-dessus des cacas avec la barre espace.

## Stack

- Angular standalone
- CSS pur
- Aucun backend

## Demarrage

Le poste courant n'a pas `node` ni `npm` dans le `PATH`, donc je n'ai pas pu executer l'installation ni le build ici.

Quand Node.js est disponible :

```bash
npm install
npm start
```

Puis ouvre `http://localhost:4200`.

## Build statique

```bash
npm run build
```

Le site sera genere dans `dist/unicorn-poop-party/`.
