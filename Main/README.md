# Cobra Kai Turbo Dojo

Static web app ultra kitsch, sans build et sans dependances. Tout fonctionne avec des fichiers statiques simples.

## Contenu

- `index.html`
- `styles.css`
- `script.js`

## Lancer en local

Ouvre `index.html` dans un navigateur.

## Mettre sur GitHub

1. Cree un repo GitHub vide.
2. Place ces fichiers a la racine.
3. Commit puis push sur `main`.
4. Active GitHub Pages depuis la branche `main` et le dossier racine.

## Notes

- La musique 8-bit est generee avec Web Audio, donc aucun MP3 ou asset externe n'est requis.
- Le son necessite un clic utilisateur dans la plupart des navigateurs: utilise le bouton de la page.
